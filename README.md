# LM2596-Buck-Converter-PCB
A KiCad-based PCB design of a 12V to 5V DC-DC buck converter using the LM2596S-5 switching regulator. The project delivers a stable 5V output with input/output filtering, Schottky freewheeling diode, LED load testing, and complete design files including schematic, PCB layout, BOM, and manufacturing-ready Gerber files
